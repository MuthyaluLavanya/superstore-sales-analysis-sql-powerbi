# 📊 Superstore Sales Analysis Dashboard | SQL + Power BI

## 📌 Project Overview

This project presents an end-to-end Sales Analytics solution built using **MySQL** and **Power BI**. The objective is to analyze Superstore sales data, identify business trends, and create an interactive dashboard for decision-making.

The project covers the complete data analytics workflow, including data cleaning, SQL analysis, data modeling, DAX calculations, and dashboard development.

## 💡 Key Business Insights

- Technology category generated the highest sales.
- Sales performance varied across different regions.
- A small number of products contributed significantly to overall revenue.
- Customer segments showed different purchasing behavior.
- Month-over-Month analysis helped identify sales growth trends.
- Profitability differed across product categories and regions.
